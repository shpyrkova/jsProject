git status - статус файлов в репозитории
git add -A - добавить все файлы в гит (после этого изменения начнут отслеживаться. аналогично "stage" файла в Source tree)
git commit -a -m"new changes" - закоммитить изменения файлов с комментом 
git push - запушить в удаленный репозиторий, если он привязан.
